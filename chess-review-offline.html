⚙ Ollama Setup

Ollama isn't responding. To enable plain-English explanations:

1. Download from ollama.com and install.

2. Pull a model (one-time ~4 GB download):

ollama pull mistral

3. In Command Prompt (not PowerShell):

set OLLAMA_ORIGINS=* ollama serve

Keep that window open. Stockfish works without Ollama.
